# Simple Student Record System (MySQL)

A fundamental relational database project designed to map student enrollments to active courses using MySQL.

## 📊 Database Schema Layout
![Database Schema](schema_diagram.png)

## 🛠️ Features Implemented
* Normalization via two related tables (`students` and `courses`).
* Data consistency managed through Primary Key and Foreign Key constraints.
* Multi-table analytical data fetching using `INNER JOIN`.

## 🚀 Execution Instructions
1. Open **MySQL Workbench**.
2. Go to `File -> Open SQL Script...` and choose `student_project.sql`.
3. Execute the script to build and seed the database instantly on your machine!
